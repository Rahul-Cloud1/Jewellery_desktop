# Kilangi Homepage

This is a modern, interactive homepage for the Kilangi Jewellery brand. The site features:

- Animated hero section
- Interactive bestsellers and collections tabs
- Animated testimonials carousel
- Smooth scroll and reveal animations
- Responsive, elegant design

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/Rahul-Cloud1/Jewellery_desktop.git
   ```
2. Open the project folder in your code editor or serve with a local web server to view in your browser.

## Project Structure

- `index.html` — Main HTML file
- `css/style.css` — All styles and animations
- `js/script.js` — All interactivity and effects
- `assets/` — Images and logo

## Customization
- Replace images in the `assets/` folder with your own product images.
- Update product data in the HTML or extend the JavaScript for dynamic content.
